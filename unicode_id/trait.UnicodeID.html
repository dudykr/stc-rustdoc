fn is_id_start(self) -> bool

Returns whether the specified character satisfies the ‘ID_Start’ Unicode property.

‘ID_Start’ is a Unicode Derived Property specified in UAX #31, mostly similar to ID_Start but modified for closure under NFKx.

source

fn is_id_continue(self) -> bool

Returns whether the specified char satisfies the ‘ID_Continue’ Unicode property.

‘ID_Continue’ is a Unicode Derived Property specified in UAX #31, mostly similar to ‘ID_Continue’ but modified for closure under NFKx.
